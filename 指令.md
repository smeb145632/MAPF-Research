```bash

```

## 四、WSL — 运行求解器

### 4.1 运行 Python 规划器（lifelong）

```bash
cd /mnt/f/MAPF/LeagueOfRobotRunners-MAPF

./build/lifelong --inputFile ./example_problems/random.domain/random_20.json -o output.json
```

### 4.2 查看帮助

```bash
./build/lifelong --help
```

### 4.3 常用参数

| 参数 | 说明 |
|------|------|
| `--inputFile <file>` | 输入 JSON 配置文件路径 |
| `-o <file>` / `--output <file>` | 输出 JSON 文件路径 |
| `--planTimeLimit <ms>` | 每步规划时间限制（毫秒） |
| `--preprocessTimeLimit <s>` | 预处理时间限制（秒） |
| `--simulationTime <t>` | 最大模拟步数 |
| `--outputScreen <1/2/3>` | 输出详细程度（1=全部，2=摘要，3=仅摘要） |

### 4.4 带时间限制运行

```bash
./build/lifelong \
  --inputFile ./example_problems/random.domain/random_20.json \
  -o output.json \
  --planTimeLimit 1000 \
  --preprocessTimeLimit 300
```

## 十三、常见问题

### Q1: 编译报错 "cannot execute: required file not found"
**原因**：`.sh` 文件带有 Windows CRLF 换行符（`\r\n`），Linux 无法识别。  
**修复**：
```bash
sed -i 's/\r$//' /mnt/f/MAPF/LeagueOfRobotRunners-MAPF/compile.sh
```
修复后再重新运行 `./compile.sh`。

### Q2: 编译报错 "pybind11.h not found"
在 WSL 里确认 pybind11 已安装：
```bash
python3 -c "import pybind11; print(pybind11.__version__)"
```

### Q3: 运行找不到 pyMAPFPlanner 模块
确认从项目根目录运行，且 `python/pyMAPFPlanner.py` 存在：
```bash
./build/lifelong --inputFile ... -o ...
```

### Q4: PlanViz 窗口不显示
GUI 程序需要在 Windows 原生环境运行，不要在 WSL 里跑。

### Q5: 输出文件路径问题
WSL 和 Windows 文件系统互通：
- WSL 路径 `/mnt/f/...` = Windows `F:\...`
- `output.json` 在 WSL 里生成，Windows 也可直接访问 `F:\...\output.json`
